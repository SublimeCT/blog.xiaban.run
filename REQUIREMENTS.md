# 需求文档

## 链接提取与处理 (extract-link)
- **目标**: 提取 Markdown 文件中的链接到文末的 "参考" 章节，并将正文中的链接转换为新窗口打开的形式。
- **规则**:
    1. 忽略以 `#` 开头的锚点链接。
    2. 正文中的 Markdown 链接 `[text](url)` 转换为 `<a href="url" target="_blank">text</a>`。
    3. 文末追加 `## 参考` 章节（如果不存在）。
    4. "参考" 章节中列出所有提取的链接，格式为 `- [text](url)`。
    5. "参考" 章节中禁止重复链接。
